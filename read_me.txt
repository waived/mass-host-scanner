/////////////////////
/// ABYSS SCANNER ///
/////////////////////

Description:
    Light-weight, multi-threaded TCP-connect scanner designed for  Windows operating systems.

Overview:
    One or more IP addresses can be loaded into Abyss Scanner via a list. The user will input
    at least one or more ports to scan. This program provides verbose output when scanning.
    Upon completion, the end-user will have the option to export all successful probes to a
    textfile.

Credits:
    This tool is based off of AngryScanner v1.0 by NyanCat (github.com/nyan-x-cat) originally
    featured on Hackforums.net (https://hackforums.net/showthread.php?tid=6065361).

Use responsibly!
